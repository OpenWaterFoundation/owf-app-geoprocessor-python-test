# ReadTableFromDelimitedFile Tests

|Test|Description|
|----|-----|
|`test-ReadTableFromDelimitedFile-csv`|Tests reading a table from a delimited file saved as a `CSV` file. Default delimiter of a `COMMA`.|
|`test-ReadTableFromDelimitedFile-Delimiter`|Tests the `DELIMITER` command parameter. Reads a table saved as a `CSV` file. Delimiter of a `SEMICOLON`.|
|`test-ReadTableFromDelimitedFile-text`|Tests reading a table from a delimited file saved as a `TEXT` file. Default delimiter of a `COMMA`.|
